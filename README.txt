User can click any playlist in the "Play List" Bar.
"My Music Tags" stores all tags that have been created by the user.
User can click the pencil icon to add/remove tags.
User can click the middle search icon to search for all songs 
with any of a given set of tags. User can also click the right search 
icon to search for all songs with all of a given set of tags.

Enhancement:
Create a playlist from the results of a search. 
Add it to the user Spotify account using the web services at your disposal.
